About Wave Interference

Create mesmerizing moiré patterns by combining multiple sine waves with different frequencies, amplitudes, and phases.

How to Use:

  • Adjust individual wave parameters using the sliders
  • Toggle waves on/off to see their contribution
  • Try different color modes for varied aesthetics
  • Use presets for quick pattern generation
  • Export your creations as images

Keyboard Shortcuts:

  • Space - Play/Pause animation
  • R - Reset all parameters
  • S - Take screenshot
  • F - Toggle fullscreen
  • 1-4 - Switch between waves
